# USER MANUAL — DUDUANG / FLORDER

## 1. ภาพรวมระบบ
ระบบ DUDUANG เป็นเว็บแอปสำหรับ:
- ดูดวงแบบ Interactive
- แนะนำสินค้าให้สอดคล้องกับผลดูดวง
- สั่งซื้อและชำระเงิน
- ติดตามคำสั่งซื้อ
- จัดการหลังบ้านสำหรับแอดมิน

บทบาทผู้ใช้งาน:
- User: ใช้งานดูดวง ซื้อสินค้า ติดตามคำสั่งซื้อ
- Admin: ตรวจสลิป อัปเดตสถานะคำสั่งซื้อ จัดการสินค้า

---

## 2. การเข้าใช้งานระบบ
Frontend: `http://localhost:3000`  
Backend API: `http://localhost:5001`

### 2.1 การสมัครสมาชิก
1. ไปที่หน้า `สมัครสมาชิก`
2. กรอกข้อมูลให้ครบ
3. กด `สมัครสมาชิก`
4. เมื่อสำเร็จ ให้เข้าสู่ระบบ

### 2.2 การเข้าสู่ระบบ
1. ไปที่หน้า `เข้าสู่ระบบ`
2. กรอกอีเมลและรหัสผ่าน
3. กด `เข้าสู่ระบบ`
4. ถ้าเป็นบัญชีแอดมิน ระบบจะพาไปหน้า `/admin` อัตโนมัติ

---

## 3. คู่มือการใช้งานฝั่ง User

### 3.1 เริ่มดูดวง
1. เข้าหน้า `ดูดวง` (`/draw`)
2. ตอบคำถามตามขั้นตอน
3. ไปหน้าเลือกไพ่ (`/choose`)
4. เลือกไพ่ 1 ใบเพื่อเปิดคำทำนาย
5. ระบบแสดงผลดูดวง พร้อมสินค้าแนะนำ

หมายเหตุ:
- หลังจบการเปิดไพ่ ระบบมีอนิเมชันเลื่อนไปยังส่วนสินค้าแนะนำอัตโนมัติ

### 3.2 ดูรายละเอียดสินค้า
1. จากหน้าร้าน `/shop` หรือหน้าสินค้าแนะนำ
2. กด `ดูสินค้า`
3. ตรวจสอบรายละเอียดสินค้าใน `/products/[id]`
4. กด `ใส่ตะกร้า`

### 3.3 ตะกร้าและสร้างคำสั่งซื้อ
1. ไปที่ `/cart`
2. ปรับจำนวนสินค้า
3. เลือกวิธีชำระเงิน
4. กด `ยืนยันคำสั่งซื้อ`

### 3.4 ชำระเงิน
หน้า `/payment` รองรับ 2 รูปแบบ:

- PromptPay
1. สแกน QR เพื่อโอนเงิน
2. กด `ไปหน้าอัปโหลดสลิป`
3. อัปโหลดสลิปที่ `/upload-slip`

- Credit Card (Mock)
1. กรอกเลขบัตร
2. ต้องกรอกครบ 16 หลัก
3. กด `ยืนยันการชำระเงิน`

ผลลัพธ์:
- สำเร็จ: `/pay-success`
- ไม่สำเร็จ: `/pay-fail`

### 3.5 ตรวจสอบคำสั่งซื้อ
1. ไปที่ `/orders`
2. ดูสถานะคำสั่งซื้อทั้งหมด
3. กดดูรายละเอียดใน `/orders/[id]`
4. ถ้าออเดอร์ยังค้างชำระ สามารถกลับไปชำระต่อได้

### 3.6 จัดการโปรไฟล์
1. ไปที่ `/profile`
2. แก้ไขชื่อ ที่อยู่ เบอร์โทร
3. เปลี่ยนอีเมลหรือรหัสผ่านได้
4. เมื่อกดบันทึกและมีการเปลี่ยนข้อมูลสำคัญ ระบบจะแสดง Popup ยืนยันก่อนบันทึก

### 3.7 ประวัติดูดวง
1. ไปที่ `/fate-history`
2. ดูคำทำนายย้อนหลัง
3. กดเข้าสินค้าแนะนำจากผลเดิมได้

---

## 4. คู่มือการใช้งานฝั่ง Admin

### 4.1 หน้าแดชบอร์ด `/admin`
ใช้งานหลัก:
- ดูสถิติภาพรวม
- ดูรายการรอตรวจสลิป
- ดูรายการคำสั่งซื้อ
- อัปเดตสถานะคำสั่งซื้อ

### 4.2 ตรวจสอบสลิป
1. เปิดแท็บ `ตรวจสลิป`
2. ตรวจรายการที่อัปโหลด
3. กดอนุมัติสลิป
4. ระบบปรับสถานะคำสั่งซื้อให้อัตโนมัติ

### 4.3 จัดการคำสั่งซื้อ
1. เปิดแท็บ `คำสั่งซื้อ`
2. ตรวจรายการทั้งหมด
3. อัปเดตสถานะตามขั้นตอน:
- paid
- preparing
- shipping
- completed
- cancelled

### 4.4 จัดการสินค้า `/admin/products`
ฟังก์ชัน:
- เพิ่มสินค้าใหม่
- แก้ไขสินค้า
- ลบสินค้า
- เปิด/ปิดการขาย
- ปรับสต็อกเพิ่ม/ลด
- ค้นหาสินค้า

## 7. เช็กลิสต์ทดสอบก่อนส่งงาน
- สมัครสมาชิกได้
- ล็อกอิน user/admin ได้
- ดูดวงครบ flow
- เพิ่มสินค้าเข้าตะกร้าและสร้างออเดอร์ได้
- ชำระแบบ PromptPay และอัปโหลดสลิปได้
- ชำระแบบ Credit Card โดยกรอกเลขครบ 16 หลักได้
- User เห็นประวัติคำสั่งซื้อและประวัติดูดวง
- Admin ตรวจสลิปและอัปเดตสถานะได้
- Admin จัดการสินค้าได้ครบ (เพิ่ม/แก้ไข/ลบ/สต็อก/active)
